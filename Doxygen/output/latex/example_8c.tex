A simple stub function to show how links do work. 

Links are generated automatically for webpages (like \href{http://www.google.co.uk}{\tt http\+://www.\+google.\+co.\+uk}) and for structures, like \hyperlink{struct_box_struct__struct}{Box\+Struct\+\_\+struct}. For typedef-\/ed types use \hyperlink{example_8c_a718548848862b3d5da8f7c7cea677de1}{Box\+Struct}. For functions, automatic links are generated when the parenthesis () follow the name of the function, like \hyperlink{example_8c_aaddf8ced9406a68229baee8c1e922b6a}{Box\+\_\+\+The\+\_\+\+Function\+\_\+\+Name()}. Alternatively, you can use \hyperlink{example_8c_aaddf8ced9406a68229baee8c1e922b6a}{Box\+\_\+\+The\+\_\+\+Function\+\_\+\+Name}. \begin{DoxyReturn}{Rückgabe}
